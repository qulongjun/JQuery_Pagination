基于JQuery的分页页码生成插件Paging.js
V1.3-final版本
==============================================================================================================
使用方法：导入JQuery文件，并导入paging-V1.3-final.js文件（压缩版为paging-V1.3-final.min.js）。调用方法为：父容器.paging(object)
参数列表：
1.pageCount（必需）：页码总数
2.initPage：初始页码位置
3.showCount：每次显示多少页
4.fn：回调函数，参数为当前页码数
5.liCss：默认页码样式，JSON格式
6.liActive：选中时页码的样式，JSON格式
7.liForbid：不可用时页码的样式,JSON格式

具体示例见Demo文件
